# tracking-robot
## Tracking Camera 
**Requirment** : 
 >-Arduino
 >-raspberrypi zero w
 >-camera
 >-laptop or pc
 ****
 ## Instruction
 - upload "arduino.ion"  to aurdino
	 - connect pin 5 to up-down servo
	 - connect pin 6 to right left servo
 - run "com.py" file in pc
 - run "pi.py" file in raspberry pi
	 - connect ground and vcc pin of both motor to raspberry pi
